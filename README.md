# Analysis
Each query investigates a specific aspect of the data analyst job market in relation to remote jobs:

## 1. Top Paying Data Analyst Jobs
To identify the highest-paying roles, I filtered data analyst positions by average yearly salary and location, focusing on remote jobs, to find the top 10 highest-paying positions.

```sql
SELECT  
    job_id
,   job_title
,   job_location
,   job_schedule_type
,   salary_year_avg
,   job_posted_date
,   cd.name AS company_name
FROM job_postings_fact jpf
LEFT JOIN company_dim cd ON cd.company_id = jpf.company_id
WHERE jpf.job_title_short = 'Data Analyst'
    AND jpf.salary_year_avg IS NOT NULL
    AND jpf.job_work_from_home = TRUE
ORDER BY jpf.salary_year_avg DESC
LIMIT 10;
```
There's a wide variety of companies, job titles, and a wide salary range. Unsurprisingly, the top 10 jobs appear to be leadership roles with multiple 'principle' and 'director' roles.

## 2. Skills for Top Paying Jobs
Using a CTE to gather results for the top 10 paying jobs, I joined with the skills table to identify the skills being mentioned in these job postings.
```sql
WITH top_paying_jobs AS (
    SELECT  
        job_id
    ,   job_title
    ,   salary_year_avg
    ,   cd.name AS company_name
    FROM job_postings_fact jpf
    LEFT JOIN company_dim cd ON cd.company_id = jpf.company_id
    WHERE jpf.job_title_short = 'Data Analyst'
        AND jpf.salary_year_avg IS NOT NULL
        AND jpf.job_work_from_home = TRUE
    ORDER BY jpf.salary_year_avg DESC
    LIMIT 10
)

SELECT top_paying_jobs.*
,      skills
FROM top_paying_jobs
INNER JOIN skills_job_dim sjd ON top_paying_jobs.job_id = sjd.job_id
INNER JOIN skills_dim sd ON sjd.skill_id = sd.skill_id
```

## 3. In-Demand Skills for Data Analysts
Similar to the second part of the query for Q2, the aim of this query is to find the most in-demand skills for Data Analysts. This time, I wrote a query to count instances of each skill across all remote Data Analyst jobs in the dataset. I tightened the result set to show the top 5 due to the drop-off in instances.
```sql
SELECT sd.skills, COUNT(sjd.job_id) AS skill_demand
FROM job_postings_fact jpf
INNER JOIN skills_job_dim sjd ON jpf.job_id = sjd.job_id
INNER JOIN skills_dim sd ON sjd.skill_id = sd.skill_id
WHERE jpf.job_title_short = 'Data Analyst' AND
      jpf.job_work_from_home = TRUE
GROUP BY sd.skills
ORDER BY skill_demand DESC
LIMIT 5
```
We can see that the results have some overlap with Q2, specifically in regards to SQL, Python, Excel, and Tableau.

|----------|--------------|
| sql      | 7291         |

## 4. Skills Based on Salary
Now we know the high demand skills, but does this correlate with the highest-paying skills? To answer this, I calculated the average salaries associated with each skill.
```sql
SELECT sd.skills, ROUND(AVG(jpf.salary_year_avg), 0) AS average_salary
FROM job_postings_fact jpf
INNER JOIN skills_job_dim sjd ON jpf.job_id = sjd.job_id
INNER JOIN skills_dim sd ON sjd.skill_id = sd.skill_id
WHERE jpf.job_title_short = 'Data Analyst' AND
      jpf.job_work_from_home = TRUE AND
      jpf.salary_year_avg IS NOT NULL
GROUP BY sd.skills
ORDER BY average_salary DESC
LIMIT 25
```
The results here diverge from the results of the previous two queries, so I widened the result set to return the top 25 to confirm. ## 5. Most Optimal Skills to Learn
Using the results from the previous questions, can we now determine which skills are most 'optimal' to learn? To answer this, I combined the two previous queries.
```sql
WITH skills_demand AS (
    SELECT sd.skill_id, sd.skills, COUNT(sjd.job_id) AS skill_demand
    FROM job_postings_fact jpf
    INNER JOIN skills_job_dim sjd ON jpf.job_id = sjd.job_id
    INNER JOIN skills_dim sd ON sjd.skill_id = sd.skill_id
    WHERE jpf.job_title_short = 'Data Analyst' AND
        jpf.job_work_from_home = TRUE AND
        jpf.salary_year_avg IS NOT NULL
    GROUP BY sd.skill_id
) 
, average_salary AS (
    SELECT sd.skill_id, sd.skills, ROUND(AVG(jpf.salary_year_avg), 0) AS average_salary
    FROM job_postings_fact jpf
    INNER JOIN skills_job_dim sjd ON jpf.job_id = sjd.job_id
    INNER JOIN skills_dim sd ON sjd.skill_id = sd.skill_id
    WHERE jpf.job_title_short = 'Data Analyst' AND
        jpf.job_work_from_home = TRUE AND
        jpf.salary_year_avg IS NOT NULL
    GROUP BY sd.skill_id
)

SELECT skills_demand.skill_id
,      skills_demand.skills
,      skill_demand
,      average_salary
FROM skills_demand
INNER JOIN average_salary ON skills_demand.skill_id = average_salary.skill_id
ORDER BY skill_demand DESC, average_salary DESC
LIMIT 25
```

# Project Conclusions
This project analysed remote Data Analyst job postings to answer five questions about salaries, skills, and market demand. The key takeaways are:
- Ignoring some outliers (the 'Data Analyst' who earns over 600k), the top 10 highest-paying remote Data Analyst roles show high earning potential — but these represent a thin upper tail of the market, often tied to senior or specialised titles at large tech firms. They are useful as aspirational benchmarks but poor guides for what a typical job search will yield.
- The skills behind top salaries skew specialised. Q2 revealed that the highest-paying roles cluster around a core of SQL and Python, but also require tools like Snowflake, Pandas, and cloud platforms — suggesting that salary uplift at the top end comes from pairing fundamentals with in-demand technical depth.
- Skills such as SQL, Python, and Excel, along with some form of visualisation tool such as Power Bi or Tableau, remain the foundation and an aspiring analyst should treat these as non-negotiable.
- The highest-paying skills are often niche. Q4 identified skills — such as less common programming languages or specialised platforms — commanding the highest average salaries. However, these outliers reflect scarcity rather than broad demand, and chasing them without a solid foundation carries risk.
- The optimal skills balance demand with pay. Q5 combined demand and salary to surface the most strategically valuable skills. These clustered around similar average salaries with no significant outliers, making them a more grounded picture of what the market actually needs — a useful contrast to the extremes surfaced in Q1 and Q4.
- In short, an aspiring Data Analyst should build a foundation in SQL, Python, and Excel first, along with Tableau or Power BI for visualisation. Then they may selectively develop depth in cloud tools or statistical languages to move into a more specialised role, associated with higher salaries. The data consistently points toward this path as the highest-return investment for someone entering or growing in the field.
